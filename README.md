## input
* URL
* TYPE
* 출력묶음단위

## output
* 몫
* 나머지

## 조건
* 모든 문자 입력가능
* 영어, 숫자만 출력
    * html
    * text
* 오름차순
    * 숫자 : 0, 1, 2 ..
    * 영어 : AaBb ..... YyZz
* 영어, 숫자 Mix 교차 출력
* 영어, 숫자, 영어, 숫자 ....
* 출력 묶음 단위
    * input(출력묶음단위) 값 을 참고하여 몫과 나머지 문자를 따로 처리
